Ex. 008 - Texto em negrito

Negrito / Destaque

Nesta frase há um termo em negrito utilizando a tag <b>.

Já nessa, há um termo em destaque utilizando a tag <strong>.

Itálico / Ênfase

Nesta frase, temos um termo itálico utilizando a tag <i>.

Já nessa, temos um termo em ênfase utilizando a tag <em> (tag semântica para html5).

Texto marcado

Podemos criar também um texto marcado utilizando a tag <mark>.

Para alterar a cor do texto marcado, utilizamos CSS adicionando na tag o código "style" + nome da cor.

Exemplo: texto marcado em roxo.

Texto pequeno

Nesta frase podemos escrever palavras pequenas utilizando a tag <small>.

Texto deletado

Podemos utilizar marcar um texto como excluído para indicar que ele deve ser lido, porém não considerado utilizando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dá-lo ênfase e indicar que ele foi adicionado depois utilizando a tag <ins>.

Texto sobrescrito

x2+3

utilizando a tag <sup>.

Texto subescrito

H2O

utilizando a tag <sub>.
